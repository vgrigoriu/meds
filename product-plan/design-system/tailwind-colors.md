# Tailwind Color Usage

This app uses Tailwind's built-in color palettes. No custom colors needed.

## Primary: Teal

Use for interactive elements, accents, and emphasis.

| Use Case | Light Mode | Dark Mode |
|----------|------------|-----------|
| Buttons | `bg-teal-500 hover:bg-teal-600` | `bg-teal-500 hover:bg-teal-400` |
| Links | `text-teal-600 hover:text-teal-700` | `text-teal-400 hover:text-teal-300` |
| Focus rings | `ring-teal-500` | `ring-teal-500` |
| Selected states | `bg-teal-50` | `bg-teal-950/30` |
| Badges | `bg-teal-100 text-teal-700` | `bg-teal-900/40 text-teal-300` |

## Secondary: Amber

Use for warnings and highlighting (expired medications).

| Use Case | Light Mode | Dark Mode |
|----------|------------|-----------|
| Warning bg | `bg-amber-50` | `bg-amber-950/30` |
| Warning text | `text-amber-700` | `text-amber-300` |
| Warning border | `border-amber-200` | `border-amber-800` |
| Expired highlight | `bg-amber-50 border-amber-200` | `bg-amber-950/20 border-amber-800` |

## Neutral: Slate

Use for backgrounds, text, borders, and cards.

| Use Case | Light Mode | Dark Mode |
|----------|------------|-----------|
| Page bg | `bg-slate-50` | `bg-slate-950` |
| Card bg | `bg-white` | `bg-slate-900` |
| Primary text | `text-slate-900` | `text-slate-100` |
| Secondary text | `text-slate-600` | `text-slate-400` |
| Muted text | `text-slate-500` | `text-slate-500` |
| Borders | `border-slate-200` | `border-slate-700` |
| Dividers | `divide-slate-100` | `divide-slate-800` |

## Error: Red

Use for error states and destructive actions.

| Use Case | Light Mode | Dark Mode |
|----------|------------|-----------|
| Error bg | `bg-red-50` | `bg-red-950/30` |
| Error text | `text-red-700` | `text-red-300` |
| Error border | `border-red-200` | `border-red-800` |
